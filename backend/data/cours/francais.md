## Leçon 1 : Grammaire — Les Natures de Mots

### 1.1 Les classes grammaticales

| Classe                    | Rôle                                      | Exemples                  |
| ------------------------- | ----------------------------------------- | ------------------------- |
| **Nom**                   | Désigne un être, une chose, un concept    | chien, liberté, Abidjan   |
| **Article**               | Détermine le nom                          | le, la, les, un, une, des |
| **Adjectif qualificatif** | Qualifie ou précise le nom                | beau, grand, intelligent  |
| **Pronom**                | Remplace un nom                           | il, elle, je, celui-ci    |
| **Verbe**                 | Exprime une action ou un état             | courir, être, avoir       |
| **Adverbe**               | Modifie un verbe, adjectif, autre adverbe | vite, très, bien          |
| **Préposition**           | Relie des éléments                        | à, de, dans, sur, avec    |
| **Conjonction**           | Relie des mots ou propositions            | et, mais, ou, donc, car   |
| **Interjection**          | Exprime un sentiment                      | Ah !, Hélas !, Bravo !    |

---

### 1.2 Les fonctions grammaticales

**Sujet :** de qui ou de quoi parle-t-on ? (Pose la question : Qui est-ce qui ? Qu'est-ce qui ?)

**Verbe :** action ou état du sujet.

**Complément d'objet direct (COD) :** répond à "qui ?" ou "quoi ?" après le verbe, sans préposition.

> _Il mange **une mangue**._ (COD = une mangue)

**Complément d'objet indirect (COI) :** répond à "à qui ?", "de qui ?", "à quoi ?", "de quoi ?", avec préposition.

> _Il parle **à son ami**._ (COI = à son ami)

**Attribut du sujet :** relié au sujet par un verbe d'état (être, paraître, sembler, devenir...).

> _Il **est médecin**._

**Épithète :** adjectif directement lié au nom.

> _Une belle **journée**._

**Apposition :** nom ou groupe nominal placé à côté du nom pour le préciser, séparé par une virgule.

> _Abidjan, **capitale économique**, est une ville animée._

## Leçon 2 : Conjugaison

### 2.1 Les groupes de verbes

| Groupe     | Caractéristique                                | Exemples                         |
| ---------- | ---------------------------------------------- | -------------------------------- |
| 1er groupe | Infinitif en **-er**                           | aimer, manger, partir            |
| 2e groupe  | Infinitif en **-ir** + gérondif en **-issant** | finir (finissant), choisir       |
| 3e groupe  | Tous les autres                                | être, avoir, aller, venir, faire |

---

### 2.2 Temps principaux

#### Indicatif présent (1er groupe — ex : parler)

| Pronom    | Terminaison | Exemple      |
| --------- | ----------- | ------------ |
| Je        | -e          | je parle     |
| Tu        | -es         | tu parles    |
| Il/Elle   | -e          | il parle     |
| Nous      | -ons        | nous parlons |
| Vous      | -ez         | vous parlez  |
| Ils/Elles | -ent        | ils parlent  |

#### Passé composé

```
Auxiliaire AVOIR ou ÊTRE (au présent) + Participe passé

J'ai parlé. / Je suis allé(e).
```

**Verbes conjugués avec ÊTRE :**  
aller, venir, partir, arriver, entrer, sortir, naître, mourir, rester, tomber, monter, descendre, passer, retourner (+ leurs composés) + tous les verbes pronominaux.

**Accord du participe passé :**

- Avec ÊTRE → s'accorde avec le sujet
- Avec AVOIR → s'accorde avec le COD si celui-ci est **avant** le verbe

#### Imparfait

```
Radical du verbe à la 1ère pers. du pluriel (présent) + terminaisons :
-ais, -ais, -ait, -ions, -iez, -aient
```

Exemple : nous parlons → parl- → je parlais, tu parlais...

#### Futur simple

```
Infinitif du verbe + terminaisons :
-ai, -as, -a, -ons, -ez, -ont
```

Exemple : je parlerai, tu parleras, il parlera...

**Futur des verbes irréguliers :**

| Verbe   | Futur       |
| ------- | ----------- |
| être    | je serai    |
| avoir   | j'aurai     |
| aller   | j'irai      |
| faire   | je ferai    |
| venir   | je viendrai |
| voir    | je verrai   |
| pouvoir | je pourrai  |
| vouloir | je voudrai  |
| savoir  | je saurai   |

---

### 2.3 Le subjonctif présent

**Emploi :** après des verbes exprimant le doute, la volonté, la nécessité, le sentiment, la crainte.

```
Expressions déclenchant le subjonctif :
- Il faut que...
- Je veux que...
- Je souhaite que...
- Bien que...
- Pour que...
- Avant que...
- À condition que...
```

**Formation :** radical de la 3ème personne du pluriel (présent) + terminaisons : -e, -es, -e, -ions, -iez, -ent.

## Leçon 3 : Orthographe

### 3.1 Les accords fondamentaux

**Accord du nom et de l'adjectif :**

- Un adjectif s'accorde en **genre** (masculin/féminin) et en **nombre** (singulier/pluriel) avec le nom auquel il se rapporte.

| Masculin singulier | Féminin singulier | Masculin pluriel | Féminin pluriel |
| ------------------ | ----------------- | ---------------- | --------------- |
| grand              | grande            | grands           | grandes         |
| beau               | belle             | beaux            | belles          |
| vieux              | vieille           | vieux            | vieilles        |

**Pluriel des noms :**

- Règle générale : + **s** (livre → livres)
- Noms en -eau : + **x** (bateau → bateaux)
- Noms en -al : → **aux** (cheval → chevaux, sauf : bal, carnaval, festival → bals, carnavals, festivals)
- Noms en -ail : + **s** ou → **aux** (travail → travaux, détail → détails)

---

### 3.2 Homophones grammaticaux

| Homophone | Nature                   | Astuce                                |
| --------- | ------------------------ | ------------------------------------- |
| **a**     | verbe avoir              | Peut être remplacé par "avait"        |
| **à**     | préposition              | Ne peut PAS être remplacé par "avait" |
| **est**   | verbe être               | Peut être remplacé par "était"        |
| **et**    | conjonction              | Peut être remplacé par "et puis"      |
| **ou**    | conjonction de choix     | Peut être remplacé par "ou bien"      |
| **où**    | pronom/adverbe de lieu   | Indique un lieu ou un choix temporel  |
| **son**   | déterminant possessif    | Peut être remplacé par "mon" ou "ton" |
| **sont**  | verbe être               | Peut être remplacé par "étaient"      |
| **ces**   | déterminant démonstratif | Remplaçable par "mes/les"             |
| **ses**   | déterminant possessif    | Remplaçable par "mes"                 |
| **c'est** | présentatif              | = ceci est                            |
| **s'est** | pronom + auxiliaire      | Verbe pronominal au passé             |

---

### 3.3 Difficultés orthographiques courantes

**Verbes en -cer et -ger :**

- -cer : prend une cédille devant a/o → nous commençons, il avançait
- -ger : prend un e devant a/o → nous mangeons, il nageait

**-quoi / -quet / -quer :** pour le son [k] devant e et i → manquer → nous manquons ✓

**Mots invariables à connaître :**

```
néanmoins, cependant, toutefois, davantage, plutôt,
désormais, dorénavant, quoique, quoiqu'il en soit,
voire, nonobstant, certes, jadis, naguère
```

## Leçon 4 : Stylistique et Figures de Style

### 4.1 Les figures de style principales

**Comparaison :** rapprochement entre deux éléments grâce à un outil de comparaison (comme, tel, ainsi que...).

> _Il est courageux **comme un lion**._

**Métaphore :** comparaison sans outil de comparaison.

> _C'est **un lion** sur le terrain._

**Personnification :** on prête des attributs humains à un objet, animal ou abstraction.

> _La mer **chuchote** ses secrets._

**Hyperbole :** exagération pour créer un effet.

> _Je t'ai attendu **des millénaires**._

**Litote :** dire moins pour suggérer plus.

> _Ce n'est pas mal du tout._ (= c'est très bien)

**Antiphrase :** dire le contraire de ce qu'on pense (ironie).

> _Bravo, c'est du beau travail !_ (= c'est raté)

**Anaphore :** répétition d'un mot ou groupe en début de phrase.

> _Je refuse. Je refuse de me taire. Je refuse de capituler._

**Chiasme :** disposition en miroir (ABBA).

> _Il faut manger pour vivre et non vivre pour manger._

**Euphémisme :** atténuation d'une réalité pénible.

> _Il a **quitté ce monde**._ (= il est mort)

## Leçon 5 : Types et Genres de Textes

### 5.1 Les types de textes

| Type             | Objectif                       | Marqueurs                                      |
| ---------------- | ------------------------------ | ---------------------------------------------- |
| **Narratif**     | Raconter une histoire          | Passé simple, imparfait, connecteurs temporels |
| **Descriptif**   | Décrire un lieu, une personne  | Imparfait, adjectifs, adverbes de lieu         |
| **Argumentatif** | Convaincre, défendre une thèse | Connecteurs logiques, exemples, réfutation     |
| **Explicatif**   | Faire comprendre               | Présent, définitions, exemples                 |
| **Injonctif**    | Donner des ordres, conseils    | Impératif, infinitif, devoir + infinitif       |

---

### 5.2 Les connecteurs logiques

| Relation         | Connecteurs                                                 |
| ---------------- | ----------------------------------------------------------- |
| **Addition**     | de plus, en outre, également, par ailleurs, de surcroît     |
| **Opposition**   | mais, cependant, toutefois, néanmoins, or, en revanche      |
| **Cause**        | car, parce que, puisque, en effet, vu que, attendu que      |
| **Conséquence**  | donc, ainsi, c'est pourquoi, par conséquent, dès lors       |
| **Concession**   | bien que, même si, quoique, certes, il est vrai que         |
| **Illustration** | par exemple, ainsi, notamment, c'est le cas de              |
| **Conclusion**   | en conclusion, pour conclure, en somme, bref, en définitive |

## Leçon 6 : Expression Écrite

### 6.1 La dissertation

**Structure :**

```
I. INTRODUCTION
   - Accroche (citation, fait, question)
   - Présentation du sujet/thème
   - Problématique (question centrale)
   - Annonce du plan

II. DÉVELOPPEMENT (2 ou 3 parties)
    Chaque partie :
    - Idée directrice (thèse)
    - Argument(s) + Exemple(s)
    - Transition

III. CONCLUSION
    - Synthèse des arguments
    - Réponse à la problématique
    - Ouverture (perspective plus large)
```

---

### 6.2 Le commentaire de texte

**Étapes :**

1. **Lecture attentive** : identifier le genre, l'époque, le registre
2. **Identification des axes de lecture** : idées principales à analyser
3. **Rédaction des axes** : chaque axe = une partie avec sous-parties (procédés + effets)
4. **Conclusion** : bilan + ouverture

**Registres littéraires :**

| Registre    | Caractéristiques                 | Exemple           |
| ----------- | -------------------------------- | ----------------- |
| Lyrique     | Émotions intenses, 1ère personne | Poésie romantique |
| Épique      | Héros, grandeur, aventure        | Homère, épopées   |
| Tragique    | Fatalité, destin inéluctable     | Racine, Sophocle  |
| Comique     | Humour, rire                     | Molière           |
| Satirique   | Critique par la dérision         | Voltaire          |
| Pathétique  | Pitié, compassion                | Zola              |
| Fantastique | Surnaturel, doute                | Maupassant        |

---

### 6.3 La correspondance formelle

**Structure d'une lettre officielle :**

```
[Lieu, Date]
[Nom de l'expéditeur]
[Adresse]

[Destinataire]
[Fonction/Titre]
[Adresse]

Objet : [en quelques mots]
[P.J. : si pièces jointes]

Madame/Monsieur [+ titre éventuel],

[Corps de la lettre :]
  - Paragraphe d'introduction (motif de la lettre)
  - Développement (arguments, faits, demande)
  - Conclusion (formule de politesse préparée)

Dans l'attente de votre réponse, je vous prie d'agréer,
Madame/Monsieur, l'expression de mes salutations distinguées.

[Signature]
```

## Leçon 7 : Littérature Africaine et Francophone

### 7.1 La Négritude

Mouvement littéraire et culturel né dans les années 1930, affirmant la valeur des cultures africaines et noires face à l'assimilation coloniale.

**Fondateurs :**

- **Aimé Césaire** (Martinique) — _Cahier d'un retour au pays natal_
- **Léopold Sédar Senghor** (Sénégal) — _Chants d'ombre_
- **Léon-Gontran Damas** (Guyane)

**Thèmes :** identité africaine, mémoire de l'esclavage, dignité, tradition orale.
